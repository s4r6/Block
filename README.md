## プロジェクト概要
**Block** は、Unityで開発したシンプルな**ブロック崩しゲームのデモ**です。  
本プロジェクトは、**SOLID原則**を知らない人に具体的な実装例を示すために制作されました。  
実際のコードを通じて、設計原則の理解を深めることを目的としています。

---

## 使用している主な技術
- **ゲームエンジン**: Unity 2022.3.22f1
- **プログラミング言語**: C#
- **設計指針**: SOLID原則（理解促進用の例）
